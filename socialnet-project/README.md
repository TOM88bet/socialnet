# SocialNet Project

A simple beginner-friendly PHP social network starter structure for a university mock project.

## Stack

- PHP
- MySQL
- Nginx
- Linux

## Project Structure

- `admin/newuser.php`
- `socialnet/signin.php`
- `socialnet/index.php`
- `socialnet/setting.php`
- `socialnet/profile.php`
- `socialnet/about.php`
- `socialnet/signout.php`
- `socialnet/menubar.php`
- `includes/db.php`
- `includes/auth.php`
- `includes/session.php`
- `assets/style.css`
- `db.sql`

## Database

- Database name: `socialnet`
- Table name: `account`
- Fields: `id`, `username`, `fullname`, `password`, `description`

## Notes

This starter project only contains the folder structure and minimal placeholder code.
